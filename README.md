# Geometry Dash Bucket List

## Setup

It is technically possible to run this project on a regular server, but I don't know how to do that and used Node.js for it. Do what you like, but I only have instructions for my own method.

1. Install Node.js under <https://nodejs.org/>. If not installed, install an Integrated Development Environment (IDE) and git. I recommend VSCode as your IDE.
2. Open VSCode or other IDE of your choice
3. Clone repository using "git clone https://github.com/Yooli8537/Geometry-Dash-Bucket-List"
4. Run the following commands in order:
npm init -y
npm install express
node server.js
